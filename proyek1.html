Instalasi IP Potion NetFlow

Saya melakukan instalasi dan konfigurasi IP Potion NetFlow sebagai solusi pemantauan lalu lintas jaringan menggunakan teknologi NetFlow. Proyek ini bertujuan untuk meningkatkan visibilitas terhadap aktivitas jaringan dan membantu tim dalam mengidentifikasi serta menganalisis masalah performa jaringan yang muncul.

Langkah-langkah yang saya lakukan dalam proyek ini termasuk:

  • Instalasi perangkat lunak IP Potion di server.
  • Konfigurasi NetFlow pada perangkat jaringan (Router dan Switch).
  • Pemantauan dan analisis lalu lintas jaringan melalui antarmuka pengguna IP Potion.
  • Pembuatan laporan performa jaringan berdasarkan hasil pemantauan NetFlow.

Instalasi Grafana, Prometheus, dan SNMP-exporter untuk Monitoring Mikrotik

Proyek ini berfokus pada implementasi sistem pemantauan jaringan menggunakan Grafana, Prometheus, dan SNMP-exporter untuk perangkat Mikrotik. Saya mengonfigurasi setiap komponen agar bekerja bersama-sama dalam memberikan visualisasi data jaringan secara real-time.

Beberapa langkah yang dilakukan dalam proyek ini:

  • Instalasi Grafana sebagai platform visualisasi.
  • Konfigurasi Prometheus untuk mengumpulkan data dari SNMP-exporter.
  • Setup SNMP-exporter pada perangkat Mikrotik untuk pengambilan data performa jaringan.
  • Mendesain dashboard di Grafana untuk menampilkan data secara real-time.

Instalasi Server MRTG dengan VirtualBox dan Ngrok

Dalam proyek ini, saya melakukan instalasi server MRTG (Multi Router Traffic Grapher) menggunakan VirtualBox sebagai platform virtualisasi dan Ngrok untuk membuat server MRTG dapat diakses melalui internet. MRTG digunakan untuk memantau lalu lintas jaringan secara visual.

Berikut adalah beberapa langkah utama dalam proyek ini:

  • Instalasi VirtualBox dan pembuatan mesin virtual untuk server MRTG.
  • Konfigurasi MRTG pada mesin virtual untuk memantau lalu lintas jaringan dari perangkat jaringan.
  • Setup Ngrok untuk memberikan akses publik ke server MRTG melalui internet.
  • Pembuatan laporan visual lalu lintas jaringan melalui antarmuka MRTG.
